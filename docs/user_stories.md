# User Stories - Lawhi

## Epic 1: Authentication & Profile
- **US 1.1**: As a user, I want to create an account with my email and password so that I can sync my data across devices.
- **US 1.2**: As a user, I want to log in to my existing account to retrieve my saved reading positions.
- **US 1.3**: As a guest, I want to use the app without an account to try the features locally.

## Epic 2: Reading Experience (Thumn/Lawh)
- **US 2.1**: As a user, I want to view the Quran text in a "Lawh" style (1 Thumn per page) to follow the Moroccan tradition.
- **US 2.2**: As a visually impaired user, I want to increase the text size so that I can read clearly without strain.
- **US 2.3**: As a user, I want to switch between different pastel backgrounds to find the most comfortable visual setting.
- **US 2.4**: As a user, I want the text to scroll automatically at my chosen speed so I can read hands-free.

## Epic 3: Progress Tracking
- **US 3.1**: As a Hifz student, I want to save my current "Hifz site" (position) with one tap so I know where to resume tomorrow.
- **US 3.2**: As a reader, I want quick-access buttons for my 4 markers (Learning, Review, Reading, Browsing) on the home screen.

## Epic 4: Interactive Tools
- **US 4.1**: As a user, I want to search for a specific Ayah by typing an Arabic keyword so I can find it quickly.
- **US 4.2**: As a user, I want to listen to a specific Ayah recited by a Qari to verify my pronunciation.

## Epic 5: Navigation
- **US 5.1**: As a user, I want to browse the Quran via a structured list of Surahs, Hizbs, and Thumns for precise navigation.
